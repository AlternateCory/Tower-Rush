Cheats
ESC = Exit Game
CTRL + S = Infinite Points
CTRL + L = Infinite Life
CTRL + W + 1-9 = Jump to wave 1-9
CTRL + R + 1-5 = Jump to room/level 1-5
Basic Info
Wave 10 from each room/level is a “Boss Level” that the player will have to beat in order to make it to the next level.
